Langkah 1:

ketika kamu menghubungi VerBotX untuk pertama kalinya, pasti kamu akan mendapatkan pesan demikian.

jangan khawatir, hal ini wajar karena kamu belum mempunyai akun VerBotX, untuk membuat akun VerBotX lihat pada langkah selanjutnya..

Langkah 2:

gunakan command atau perintah .daftar untuk melakukan pendaftaran akun VerBotX dengan contoh seperti ini :


.daftar email|nama|umur

keterangan :

  1. .daftar => command / perintah untuk melakukan pendaftaran
  2. email => diisi dengan email kamu yang masih aktif untuk menerima kode OTP
  3. nama => diisi dengan nama kamu
  4. umur => diisi dengan umur kamu

sehingga akan menjadi contoh seperti ini :

.daftar verbotx@gmail.com|verbotx|20

Langkah 3: Verifikasi Email

setelah kamu melakukan langkah diatas, kami akan mengirim verifikasi seperti ini, kamu tinggal tekan tombol "Verifikasi Sekarang" untuk melakukan verifikasi pendaftaran akun VerBotX kamu

setalah kamu menekan tombol tersebut, kamu akan langsung diarahkan ke WhatsApp VerBotX untuk mengirim kode OTP, langsung kamu kirim saja tanpa merubah apapun

Langkah 4: Penting! Jika kode OTP Tidak Masuk

Pastikan email yang kamu gunakan masih aktif ya..

jika email yang kamu gunakan aktif tetapi kode OTP tetap tidak masuk, silahkan kamu cek folder spam yang ada di gmail kamu...

Langkah 5: Selesai!

setelah kamu menerima kode OTP dan menekan tombol Verifikasi, kamu akan diarahkan langsung ke WhatsApp VerBotX untuk verifikasi akun kamu, dan jika sudah sampai titik ini maka pendaftaran kamu selesai kamu siap untuk menggunakan layanan VerBotX.
